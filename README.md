# 🌐 Aftab's Portfolio Website

## 📖 Introduction
This repository contains my personal portfolio website, designed to showcase my skills, projects, and career journey.  
The site is built with a focus on **responsive design**, clean structure, and continuous improvement to reflect my growth as a developer.

Live site: [https://im-aftab.github.io/Portfolio/](https://im-aftab.github.io/Portfolio/)

## 🛠 Tech Stack
- **Frontend**
  - HTML5 → Structure and semantic layout
  - CSS3 → Styling, responsiveness, and layout fixes
  - JavaScript → Interactivity and dynamic elements

- **Deployment**
  - Git & GitHub → Version control and collaboration
  - GitHub Pages → Hosting and live deployment

- **Tools**
  - Visual Studio Code → Code editing
  - Git CLI → Local repository management
  - Browser DevTools → Debugging and responsiveness testing
